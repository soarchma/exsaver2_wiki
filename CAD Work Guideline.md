# EXSAVER 2.0 웹 관제 시스템 연동을 위한 CAD 도면 작업 표준 가이드

본 가이드는 현장 도면(DXF) 데이터를 기반으로 EXSAVER 2.0 웹 대시보드(BEMS)에 기기 좌표와 연결 상태를 100% 자동 매핑하기 위해 제정된 필수 CAD 작성 규칙입니다.

기존 엑세이버 도면 자동화 프로그램의 룰을 해치지 않으며, 아주 간단한 레이어 및 객체 관리만으로 웹 시스템과의 완벽한 연동을 보장합니다.

## 1. B/B (시스템 박스) 레이어 단독 분리 (필수)

웹 시스템의 파싱 알고리즘이 엉뚱한 건축 기호나 마감선을 B/B로 오인하는 것을 방지하기 위해, B/B 객체는 반드시 독립된 레이어에 존재해야 합니다.

- **요구사항**: B/B 박스는 반드시 `3`번 레이어, 또는 `EX_BB` 등 **다른 건축 요소가 섞이지 않은 B/B 전용 단독 레이어**에 작성해 주십시오.
- **해결 방법**: 이미 타 요소와 섞여 있다면, `QSELECT` (신속 선택) 기능을 이용하여 B/B 박스만 일괄 선택한 후 전용 레이어로 이동시켜 주시면 됩니다.

## 2. B/B 객체 분해(Explode) 엄금 (필수)

B/B 객체가 낱개의 선(LINE) 4개로 분해되어 있으면 시스템이 사각형 형태를 인식하지 못하고 좌표 추출에 실패합니다.

- **요구사항**: B/B 박스는 반드시 **닫힌 다중선(`LWPOLYLINE` / `Closed: Yes`)** 형태이거나, 정식 **블록(`BLOCK` / `INSERT`)** 형태로 유지되어야 합니다. 절대 분해(Explode) 명령어를 사용하지 마십시오.
- **해결 방법**: 실수로 분해된 객체들이 있다면, 해당 객체들을 선택 후 `JOIN` (결합) 명령어를 사용하여 하나의 닫힌 폴리라인으로 복구해 주십시오.

## 3. 웹 시스템 파싱용 DXF 추출 (경량화 및 XREF 결합)

웹 시스템의 파싱 엔진(파이썬 워커)이 도면을 1초 만에 분석할 수 있도록, 불필요한 데이터가 제거된 '경량화된 DXF 파일'을 추출해야 합니다.

- **요구사항 1 (엑세이버 요소만 추출)**: 건축/설비 등 불필요한 레이어를 끄고(Off/Freeze), **오직 엑세이버 관련 요소(배선, B/B, LCU/채널 텍스트)만 화면에 남긴 상태**에서 해당 객체들만 선택하여 DXF로 내보내 주십시오. (`WBLOCK` 명령어 등을 활용하여 선택된 객체만 저장하면 파일 용량이 MB에서 KB 단위로 획기적으로 줄어듭니다.)
- **요구사항 2 (외부 참조 결합)**: 만약 타 도면을 참조한 상태로 DXF를 추출해야 한다면, 반드시 XREF 관리자 창에서 모든 외부 참조 도면을 **`결합(Bind) ➔ 삽입(Insert)`** 처리하여 단일 도면 내에 완전히 포함시켜 주십시오.

## 4. 프론트엔드 배경용 이미지(PNG) 생성 (필수)

웹 대시보드의 바닥에 깔릴 고해상도 배경 이미지(Base Map)를 CAD 시스템에서 직접 출력하여 DXF와 함께 업로드해야 합니다.

- **요구사항 1 (엑세이버 레이어 끄기)**: 웹 시스템이 동적 마커(아이콘)를 도면 위에 직접 그릴 예정이므로, 배경 도면에는 B/B 박스나 엑셀라인이 보이면 안 됩니다. PNG 출력 전 **엑세이버 관련 레이어를 모두 숨김(Off/Freeze) 처리**해 주십시오. (마커가 이중으로 겹쳐 보이는 현상을 방지합니다.)
- **요구사항 2 (건축 뼈대만 유지)**: 벽체, 문, 기둥 등 사용자가 위치를 파악하는 데 필요한 최소한의 건축 구조만 남기고, 복잡한 해치(Hatch)나 치수선은 깔끔하게 정리해 주시는 것이 좋습니다.
- **요구사항 3 (층별 분할 출력)**: 캐드의 출력(`PLOT`) 기능을 사용하여, 도면을 **1개 층 단위로 각각 크롭(Crop)**하여 고해상도(가로 2000px 이상 권장) PNG 파일로 출력해 주십시오.
- **요구사항 4 (출력 영역의 좌표 기록 및 여백 제거 - 🌟매우 중요)**: PNG 파일로 출력할 때 지정한 **'출력 영역(Window)'의 정확한 캐드 원본 좌표**가 웹 시스템의 캘리브레이션(위치 보정)에 반드시 필요합니다.

- 도면에 출력 영역을 나타내는 **직사각형 가이드 선**을 그리고, 해당 사각형의 **좌측 하단(Min X, Min Y) 좌표**와 **우측 상단(Max X, Max Y) 좌표**를 기록하여 PNG 파일과 함께 전달해 주십시오.
- **[주의]** PNG 출력(PLOT) 시, 기록된 직사각형 영역 바깥으로 **불필요한 흰색 여백(Margin)이 절대 포함되지 않도록** 영역을 꽉 채워서(Fit to paper, Center the plot) 이미지 파일로 저장해 주십시오. (여백이 생기면 웹에서 핀 위치가 전체적으로 밀리게 됩니다.)

## 5. (참고) 엑셀라인 및 텍스트 규칙 (현행 유지)

현재 제조사에서 CAD 자동화 프로그램으로 생성하시는 엑셀라인(Wire)과 텍스트(LCU, 제품명)의 품질이 매우 우수하여 웹 파싱에 완벽히 적합합니다.

- LCU 블록 앵커링, 폴리라인 직교성, 텍스트 배치 등의 기존 자동화 도면 생성 룰은 **변경 없이 그대로 유지**해 주시면 됩니다.
